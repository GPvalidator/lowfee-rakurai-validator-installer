# Low Fee Rakurai Installer

Automated installer for Solana validators using Rakurai.

## Features

- Installs Solana CLI
- Installs Rust
- Builds Rakurai validator
- Creates validator identity
- Creates vote account
- Configures Rakurai activation account
- Generates validator start script
- Creates systemd service
- Starts validator automatically

## Usage

```bash
git clone https://github.com/GPvalidator/lowfee-rakurai-installer
cd lowfee-rakurai-installer
npm install
npm start
